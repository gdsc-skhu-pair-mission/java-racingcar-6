## 입력
### n대의 자동차 생성
-[ ] 경주 할 자동차 이름(이름은 쉼표(,) 기준으로 구분) 입력
- 자동차 이름
  -[ ] 빈 문자열(null)
  -[ ] 중복
  -[ ] 5자 이하
  
### 전진 시도 횟수 입력
- [ ] 1 이상의 자연수
---------------------------------------------------------
## 전진
### 입력 받은 횟수만큼 전진 시도
-[ ] 랜덤 값 생성
  -[ ] 전진X : 0~3
  -[ ] 전진 : 4~9
-[ ] "-" 출력
-[ ] 차수별 우승자 저장
----------------------------------------------------------
## 출력
### 시작
-[ ] "경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)" 출력
-[ ] "시도할 회수는 몇회인가요?" 출력

### 차수별 실행 결과

### 최종 결과
-[ ] 우승자 여러명일 경우 쉼표(,)를 이용하여 구분
